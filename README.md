# GalaxyBlox Legacy

A Chrome extension for Roblox (v2.00 Legacy).

## Features
- New theme and UI
- Custom cursor, profile banners
- Robux-to-USD converter
- Quick Play and GalaxyHub panel

## Installation
1. Install from Chrome Web Store  
2. Refresh the Roblox website
